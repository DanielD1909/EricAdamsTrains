This is a train pack for Subway Builder which adds 38 new trains for the Eric Adams Challenge to Subway Builder using [addTrains](https://github.com/mhmoeller/subwaybuilder-addtrains/tree/V1.0-rc) by mhmoller. 

To install, simply download the folder and put it in the mods folder. Make sure to also install addTrains.

If you are interested in more trains, check out [danTrains](https://github.com/DanielD1909/danTrains) for a variety of options.
